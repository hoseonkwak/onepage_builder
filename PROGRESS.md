# 원페이지 빌더 개발 진행 상황

## 프로젝트 개요

원페이지(One-Page) 전용 템플릿 빌더. 사용자가 미리 정의된 템플릿에서 섹션 순서를 변경하고, 텍스트와 이미지를 교체하여 랜딩페이지를 완성할 수 있는 도구.

## 기술 스택

| 영역 | 기술 | 버전 |
|------|------|------|
| 프레임워크 | Nuxt 3 | 3.20.2 |
| 언어 | TypeScript | 5.x |
| 상태관리 | Pinia | 2.x |
| 스타일링 | Tailwind CSS | 3.x |
| 아이콘 | nuxt-icon | 0.6.10 |
| 드래그앤드롭 | vue-draggable-plus | 0.5.x |
| 애니메이션 | GSAP | 3.x |
| 슬라이더 | Swiper | 11.x |

| 이슈 | 원인 | 해결 방법 |
|------|------|-----------|
| Vue Router 경고: `/admin`, `/auth/login` 경로 관련 경고 | 브라우저 확장 프로그램 또는 이전 캐시 | 무시해도 됨 |
